# 코테 스터디

## 목표
이코테 책에 있는 문제를 모두 풀이합니다.

<br>

## 진행 방식
1. 매주 일요일에 Issues에 해당 주차에 풀이할 문제들이 업로드 됩니다.
  
2. Issues를 확인하고, `본인 이름 폴더`에 풀이 코드를 업로드 합니다.

3. 코드를 업로드할 때, **commit message 규칙**을 반드시 지켜주세요!

<br>

### commit message 규칙
```git
[알고리즘명] : 제목 (#issue number)
# ex) [그리디] : 큰 수의 법칙 (#1)
```
